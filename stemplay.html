ABOUT THE PROJECT

STEM PLAY is a Programming Game Platform for helping kids learning and logical improving, with the main type of game that improves programming and logical skills on the kids at the same time using thematic storytelling focused on the Sustainable Development Goals by the UN.

Available for Happy Code students and staff on Google Play and internal WebGL build.







A Level Builder is present to make possible the application of the kid's creativity and then play on a 2D platform kind game with the characters from the storytelling campaign.




Also to make the Happy Code student's and teachers' lives easier, the teaching material from daily usage and video content are available on the app with a PDF Reader(with a system of attachment files for every pdf needed) and Video Player with the progress save.
